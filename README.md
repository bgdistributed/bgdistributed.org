## BG Distributed

**BG Distributed** is a community-oriented website dedicated to open-source
distributed systems, infrastructure engineering, and orchestration.

The project follows a **learn in public** philosophy: documenting the
journey of understanding complex distributed systems while building
practical knowledge that others can benefit from.

### Topics

Current and future topics include:

-    HashiCorp Nomad
-    Distributed systems architecture
-    Consensus algorithms (Raft and related protocols)
-    Service discovery
-    Infrastructure automation
-    Linux systems administration
-    Networking
-    Observability
-    Open-source infrastructure projects

Over time, the site will also document experiments, reference material,
and open-source software developed under the BG Distributed organization.

### Website

The website is built with **Hugo** and deployed automatically through
**Cloudflare Pages**.

Source content is written in Markdown and version-controlled in this repository.

### Running locally

Clone the repository, including submodules:

    git clone --recurse-submodules https://github.com/bgdistributed/bgdistributed.org.git

Start the local development server:

    hugo server

The site will be available at:

[http://localhost:1313](http://localhost:1313)
